## Cài đặt và thiết lập môi trường

1. Thiết lập môi trường
```
pip install streamlit
```
2. Clone dự án 
```
git clone https://github.com/pysilot/ChatBot.git
cd ChatBot
```
3. Cài đặt thư viện cần thiết
```
pip install -r requirements.txt
```
